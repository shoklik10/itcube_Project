# itcube_Project

Приложение-викторина для проверки знаний по четырём темам. Отвечайте на вопросы и проходите темы до победного конца - неправильно отвеченные вопросы возвращаются снова.

---

## Скриншот

> **

---

## Возможности

- 4 темы: Математика, География, Наука, История - по 20 вопросов в каждой
- 4 варианта ответа на каждый вопрос (перемешиваются при каждом показе)
- Неправильно отвеченные вопросы возвращаются - пока не ответишь верно
- Прогресс-бар и счётчик правильных ответов
- Экран победы со статистикой: количество попыток и точность
- Тёмная тема с фиолетовым акцентом

---

## Технологии

- Python 3.
- tkinter (встроенная библиотека Python - установка не требуется)

---

## Как играть

1. На главном экране выберите тему
2. Отвечайте на вопросы, нажимая на один из 4 вариантов
3. Правильный ответ - вопрос засчитан ✅
4. Неправильный - вопрос вернётся позже ❌
5. Ответьте верно на все 20 вопросов - и увидишь экран победы 🏆

---

## Как добавить свои вопросы

Откройте main.py и найдите словарь TOPICS в начале файла. Каждая тема - список кортежей формата:

("Текст вопроса", "Правильный ответ", ["Правильный ответ", "Вариант 2", "Вариант 3", "Вариант 4"])
Пример добавления новой темы:

"🎬 Кино": [
    ("Кто срежиссировал «Начало»?", "Нолан", ["Нолан", "Спилберг", "Финчер", "Скорсезе"]),
    # ... остальные вопросы
],
